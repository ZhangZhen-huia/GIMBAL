 移动：
	左按键	上：底盘跟随云台				右按键	上：云台陀螺仪控制
		    中：底盘无力							中：云台编码值控制
			下：底盘小陀螺						下：自瞄控制
 
	左拨杆：vx，xy						右拨杆：横yaw，竖pitch
 
 发射：
		非自瞄模式：
					开启发射结构总开关：右按键在中间，同时滑轮向下转到底
					关闭发射结构总开关：滑轮向上转到底并保持一段时间
					
					开启摩擦轮：在发射总开关开启的情况下，右按键在中间，然后滑轮向上转到底，然后此时右按键还在中间的话就开启，若此时右按键不在中间就不开启
					关闭摩擦轮：右按键不在中间即可，或者总开关被关上

					开启拨弹盘：在发射总开关和摩擦轮都开启的情况下，滑轮向下滑即可，滑一下发一发，保持不动就连发
					
		自瞄模式：
					开启发射结构总开关：	右按键在下面，同时滑轮向下转到底
					关闭发射结构总开关：	滑轮向上转到底并保持一段时间
					
					开启摩擦轮：在发射总开关开启的情况下，然后滑轮向上转到底开启
					关闭摩擦轮：滑轮向下滑到底，右拨杆打向右下方关闭
					
					开启拨弹盘：在发射总开关和摩擦轮都开启的情况下，收到视觉的开火信号就开火
					关闭拨弹盘：在发射总开关和摩擦轮都开启的情况下，没有收到视觉的开火信号就停火
					
		强制开火：右拨杆打向左上方表示强制开火，若此时发射机构和摩擦轮都处于打开状态，就会发火
